## 🔄 How to Propose a Change
We use a structured process called an **RFC (Request for Comments)** to ensure everyone has a voice before we change how the platform works.

1.  **Start a Conversation:** Open a new **Issue** on GitHub and select the **RFC Template**.
2.  **Community Review:** We wait **14 days** to let everyone read and comment on the idea.
3.  **The Vote:** At least **2 Platform Owners** must give a "Thumbs Up" to move forward.
4.  **Finalize:** Once approved, the changes are updated in the system. Done!
